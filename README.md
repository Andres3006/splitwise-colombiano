# Splitwise Colombiano

## Descripción
Aplicación web para gestionar gastos compartidos entre amigos y calcular automáticamente las deudas.

## Tecnologías usadas
- Backend: Node.js + Express
- Frontend: React
- Base de datos: PostgreSQL

## Instalación

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm start
